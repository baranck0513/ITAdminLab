# Domain Controller Setup

## Objective
Deploy Windows Server 2022 as a Domain Controller and DNS Server for the securelab.local lab domain.

## Configuration
- Hostname: DC01
- Domain: securelab.local
- IP address: 192.168.56.10
- Roles installed: Active Directory Domain Services, DNS Server
